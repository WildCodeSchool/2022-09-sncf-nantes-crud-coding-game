# 2022-sncf-09-crud-coding-game
Basic back part template of the 3rd project.

# Instructions 
- Cloner le repo en local sur votre machine
- Créer une branche prenom_nom
- Votre mission est de créer au moins 2 actions du CRUD sur le fichier product.routes.js (GET, GET by ID, POST, PUT ou DELETE)
- Prenez bien le temps de configurer votre .env ainsi que votre BDD, un fichier SQL est là pour vous faire gagner du temps
- Si vous avez fini en avance, vous pouvez aller le plus loin possible
- Pensez à push votre branche à la fin de l'exercice (idéalement à chaque route fonctionnelle)
